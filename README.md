# Config link

Automatically backup config file into [config-pool](amomorning/config-pool)

## Installation

- `click`

## Features

- **Upload**
  - `cp` file from `dest` to `src`
- **Download**
  - `mv` file from `dest` to `dest+'bak'` (if exist)
  - create `src` dir
  - `cp` file from `src` to `dest`
